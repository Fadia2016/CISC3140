
To add files to GitHub repository using the command line in Linux:


Step1: open terminal

step2: Change the current working directory to your local repository.
	$ mkdir repo
	$ cd repo

step3: git clone using git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY
	$ git clone https://github.com/Fadia2016/CISC3140.git

step4: Create the file or move file to the local repository "repo"
	

step5: add the files using the command
	$ git add .

step6: Commits the tracked changes and prepares them to be pushed
	$ git commit -m "add your note here"

step7: push the changes to the repository using the following
	$ git push origin
